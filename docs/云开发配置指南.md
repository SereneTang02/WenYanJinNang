## 🚀 接下来你需要做的事情

### 步骤1：开通微信云开发

1. **打开微信开发者工具**
   - 打开你的小程序项目

2. **点击顶部工具栏的"云开发"按钮**
   - 第一次点击会提示你创建云开发环境
   - 点击"开通"

3. **创建云开发环境**
   - 环境名称：可以填 `prod`（生产环境）或 `dev`（开发环境）
   - 点击"确定"
   - 等待环境创建完成（大约需要1-2分钟）

4. **记录环境ID**
   - 创建完成后，在云开发控制台可以看到"环境ID"
   - 例如：`cloud1-xxx`
   - **重要：复制这个环境ID，后面要用**

---

### 步骤2：修改云开发环境ID

**修改 `app.js` 文件的第20行：**

```javascript
// 找到这一行（app.js 第20行）
env: 'your-env-id',  // 请替换为你的云开发环境ID

// 替换为你的实际环境ID，例如：
env: 'cloud1-xxx',
```

---

### 步骤3：创建数据库集合

1. **打开云开发控制台**
   - 在微信开发者工具中点击顶部"云开发"按钮
   - 进入控制台

2. **创建数据库集合**
   - 点击左侧菜单"数据库"
   - 点击"添加集合"按钮
   - 创建以下集合（集合名称必须完全一致）：

   ```
   ✅ users                （用户表）
   ✅ user_progress        （学习进度表）
   ✅ classes              （班级表，模块3需要）
   ✅ class_members        （班级成员表，模块3需要）
   ✅ pk_sessions          （PK任务表，模块3需要）
   ✅ pk_records           （PK记录表，模块3需要）
   ```

   **重要：现在可以先创建 `users` 和 `user_progress` 两个集合，其他的等实现模块3时再创建。**

3. **设置数据库权限**

   对于 **users** 集合：
   - 点击集合名称进入详情
   - 点击"权限设置"标签
   - 选择"自定义安全规则"
   - 粘贴以下规则：

   ```json
   {
     "read": "doc._openid == auth.openid",
     "write": "doc._openid == auth.openid"
   }
   ```

   对于 **user_progress** 集合：
   - 同样设置为：

   ```json
   {
     "read": "doc._openid == auth.openid",
     "write": "doc._openid == auth.openid"
   }
   ```

---

### 步骤4：上传云函数

1. **上传 login 云函数**
   - 在微信开发者工具左侧目录树中找到 `cloudfunctions/login` 文件夹
   - 右键点击 `login` 文件夹
   - 选择"上传并部署：云端安装依赖"
   - 等待上传完成（首次上传可能需要1-2分钟）

2. **上传 updateUserRole 云函数**
   - 同样右键点击 `cloudfunctions/updateUserRole` 文件夹
   - 选择"上传并部署：云端安装依赖"

3. **上传 syncProgress 云函数**
   - 右键点击 `cloudfunctions/syncProgress` 文件夹
   - 选择"上传并部署：云端安装依赖"

4. **验证上传成功**
   - 打开云开发控制台
   - 点击左侧"云函数"菜单
   - 应该能看到3个云函数：login、updateUserRole、syncProgress
   - 状态应该都是"部署成功"

---

### 步骤5：测试登录功能

1. **编译小程序**
   - 点击微信开发者工具顶部的"编译"按钮

2. **测试登录流程**

   **首次登录（学生）：**
   - 小程序启动后应该自动进入登录页
   - 点击"微信快捷登录"按钮
   - 授权用户信息
   - 进入角色选择页，选择"我是学生"
   - 选择年级（例如"七上"）
   - 填写学生信息（年级必填，其他选填）
   - 点击"完成"
   - 应该跳转到首页

   **首次登录（教师）：**
   - 登录页点击"微信快捷登录"
   - 角色选择页选择"我是教师"
   - 填写教师信息（学校、姓名必填）
   - 可选上传教师资格证
   - 点击"提交认证"
   - 应该跳转到首页

   **再次登录：**
   - 关闭小程序，重新打开
   - 应该自动登录，直接进入首页（不需要重新登录）

3. **验证数据是否保存**
   - 打开云开发控制台
   - 点击"数据库" → "users"
   - 应该能看到刚才注册的用户记录
   - 查看用户的 `role` 字段是否正确（student 或 teacher）

---

### 步骤6：测试退出登录

1. **进入个人中心**
   - 在小程序中进入"个人中心"页面

2. **点击退出登录**
   - 点击底部的"退出登录"按钮
   - 确认退出
   - 应该返回到登录页

3. **重新登录**
   - 再次点击"微信快捷登录"
   - 应该能直接进入首页（因为角色信息已保存）

## 🐛 常见问题排查

### 问题1：点击登录按钮没反应

**可能原因：**
- 云开发环境ID未配置
- 云函数未上传

**解决方法：**
1. 检查 `app.js` 第20行的环境ID是否正确
2. 打开云开发控制台，确认云函数已上传

### 问题2：登录时提示"登录失败"

**可能原因：**
- 云函数代码错误
- 数据库集合未创建

**解决方法：**
1. 打开云开发控制台 → 云函数 → login
2. 点击"日志"查看错误信息
3. 确认 `users` 集合已创建

### 问题3：无法授权用户信息

**可能原因：**
- 微信开发者工具的用户信息授权被禁用

**解决方法：**
1. 打开模拟器右上角的 ... 菜单
2. 选择"授权管理"
3. 找到"用户信息"，删除授权记录
4. 重新编译，再次尝试登录

### 问题4：云函数调用失败

**可能原因：**
- 云函数权限配置问题

**解决方法：**
1. 在 `project.config.json` 中确认 `cloudfunctionRoot` 配置正确：
   ```json
   "cloudfunctionRoot": "cloudfunctions/"
   ```
2. 重新上传云函数
