### 🎯 模块2：每日打卡 + 错题库

**完成度：100%**

#### 已实现功能：

**1. 练习答题页（practice）**
- ✅ 智能出题算法：
  - 每个词的每个义项生成1-2道选择题
  - 从例句中提取问题
  - 正确答案来自该义项的释义
  - 干扰项优先从同词其他义项选择
  - 题目自动洗牌
- ✅ 多种练习模式：
  - 单词练习（?wordId=1）
  - 多词练习（?wordIds=[1,2,3]）
- ✅ 答题功能：
  - 4选项界面（ABCD）
  - 即时判对错
  - 显示正确答案和解析
  - 用法说明展示
- ✅ 进度追踪：
  - 顶部进度条
  - 实时得分计算
  - 答对/答错统计
- ✅ 学习记录：
  - 自动更新 userWordProgress
  - 更新词卡颜色等级
  - 更新下次复习时间
- ✅ 完成页面：
  - 总得分展示
  - 详细统计（答对/答错/总题数）
  - 返回功能

**2. 错题库页（error-words）**
- ✅ 自动筛选错题（wrong > 0）
- ✅ 按差值倒序排序（错得最多的排最前）
- ✅ 错题详细信息：
  - 词、拼音、标签
  - 答对X次/答错Y次/差值Z
  - 颜色编码（黄色/橙色）
  - 错题徽章（需巩固/需加强）
- ✅ 三种练习模式：
  - 点击单词 → 单词专项练习
  - 全部纠错 → 练习所有错题
  - 重点突破 → 只练习差值≥3的词
- ✅ 顶部统计：
  - 错题词数量
  - 累计错误次数
- ✅ 空状态页面（无错题时显示祝贺）
- ✅ 下拉刷新

**3. 每日任务生成（utils/storage.js）**
- ✅ 智能生成每日任务：
  - 旧词选择策略：筛选到期复习的词（nextReviewAt ≤ today）
  - 新词选择策略：优先用户年级的词 + 高频词
  - 默认配置：6个旧词 + 4个新词
- ✅ 每日任务管理：
  - 生成今日任务（generateDailyTask）
  - 获取今日任务（getDailyTask）
  - 更新任务进度（updateDailyTaskProgress）
  - 获取完成进度（getDailyTaskProgress）
- ✅ 打卡记录系统：
  - 更新打卡记录（updateCheckInRecord）
  - 获取连续打卡天数（getCheckInStreak）
  - 支持365天打卡历史

**4. 数据结构**
- ✅ 用户学习进度（userWordProgress）：
  ```javascript
  {
    [wordId]: {
      correct: 5,
      wrong: 3,
      lastReviewAt: "2025-11-26T10:00:00.000Z",

  }
  ```
- ✅ 每日任务（dailyTask）：

  ```javascript
  }
  ```

  ```javascript

  ```
